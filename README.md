```
kt - keep trying 1.0
Jarrod Swart <jcswart@gmail.com>
Run a program in several threads keeping the output from the first to complete.

USAGE:
    kt [OPTIONS] [program]...

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -t, --tries <NUM_TRIES>    The number of sub threads to spawn.

ARGS:
    <program>...    The progam to run. Additional arguments are passed along.
```
